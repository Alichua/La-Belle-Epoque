Use this compact map before reading the longer `six-belle-epoques/*.md` files. For color, read `time-palettes.md` after selecting the era: each country has four 24-color palettes for morning, day, dusk, and night.

## France

- Era file: `six-belle-epoques/01-france-la-belle-epoque.md`
- Period: 1870s-1914, strongest visual core 1890s-1914.
- Default city: Paris.
- Vibe seed: electric Art Nouveau boulevard elegance with class shadows.
- Pixel anchors: Eiffel Tower silhouettes, Metro entrances, mansard roofs, cabaret glow, department-store windows, poster columns, horse cabs, bicycles, cafe terraces, gas/electric lamps, flower sellers.
- Palettes: see `time-palettes.md#france`.

## Date Search Hints

- Month/day without a year: search `site:`-free broad web queries across the era years, then refine with the strongest source.
- Full date inside the era: first try exact date, then same month/day across the era if exact results are weak.
- Full date outside the era: use the provided month/day across the era and state that adaptation.
- Favor events that can become visible props: openings, premieres, exhibitions, sports moments, political decisions, disasters, transit changes, market events, strikes, festivals, launches, or iconic cultural releases.
